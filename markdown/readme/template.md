# Test Values

## Overview

This project makes it easy to generate values for testing software.

### Scalars

Generally speaking, a method whose name is singular returns a scalar object.

Thus method call

```ruby
StringValues.string_of_size(5)
```
 
returns the ```String``` object 

```ruby
"xxxxx"
```
